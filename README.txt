В данном проекте разработали решение, которое позволит персонализировать предложения постоянным клиентам, чтобы увеличить их покупательскую активность.
Для этого было необходимо разработать модель, которая по входным признакам находила целевой - предсказывала снизиться ли покупательская активность или нет.
Для создания модели использовали модели линейной регрессии, метод k-ближайших соседей, дерево решений и метод опорных векторов. Из них выбрали лучшую, ей оказалась модель основанная на методе опорных векторов с параметрами: kernel = rbf, C = 0.1.
